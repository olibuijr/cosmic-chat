# Changelog

## 0.1.0 — 2026-07-06

### Added
- COSMIC-native IRC client built on libcosmic
- Server connection management with TLS support (rustls)
- Channel message view with text input
- Join/part/quit/action message display with colored indicators
- Nav sidebar with server and channel tree
- Connection status indicator in header bar
- Context drawer with server info and about panel
- Add-server dialog
- TOML config persistence (~/.config/cosmic-chat/config.toml)
- Desktop entry and SVG app icon for COSMIC launcher
- build.sh with local install support
